Go Render It

Simple Placeholder Images

What is this?

When creating a website mock up one of the most tedious tasks to place holder images. The original spec says 250x250 so create an image, then it changes to 150x150 or the colour changes. Then have to open the image editor and this takes time, too much time!

Go render is a service that renders images for your mock ups, all parameterised and can modified with ease. To use this change your image src to match the example below. There are also options for the colour and labels displayed.

Use in your image tags like so:

<img src="http://www.gorender.it/300/FDA782"/>

Colour Options

Go Render can take colour parameters in two formats. Either as a HEX value "FDA782" or as a RGB colour string "245,124,180". Its location within the request does not matter.

http://www.gorender.it/300x50/FDA782
http://www.gorender.it/245,124,180/300x50

Size Options

The size can be passed in two forms. Its location within the request does not matter.If a single number is passed then a square is rendered. However if "300x150" is passed then the image is rendered to the size.

http://www.gorender.it/125
http://www.gorender.it/125x125

Text Options

A custom label on the image can be passed as a parameter. Its location within the request does not matter.However the label text must begin with ":" which will be removed when the image is rendered.

http://www.gorender.it/300x50/:My Label
http://www.gorender.it/350x50/:Label/FDD432

Other options for your place holder :

  • Colour
  • Size
  • Text
